# niruthiha.github.io

Personal website of Niruthiha Selvanayagam, empirical software engineering researcher (SE4AI) at ÉTS Montréal.

Static site, no build step. Plain HTML and CSS.

- `index.html` — home (bio, education, news, publications, projects link)
- `bio.html` — longer biography
- `projects.html` — selected projects
- `style.css` — styles
- `assets/` — images

Served via GitHub Pages from the repository root.
